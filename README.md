# Velo 💰

A personal finance and wealth-building tracker built with plain HTML, CSS, and vanilla JavaScript. No frameworks were used.

---

## Features

- **Dashboard** — monthly income, expenses, net balance, and 10% savings target
- **50/30/20 Rule** — live budget tracking across needs, wants, and savings
- **Retirement Tracker** — 401(k) and Roth IRA trackers with real 2026 IRS contribution limits
- **Wealth Tab** — compound interest projections, Rule of 72, and emergency fund tracker
- **10-Year Projection Chart** — visualizes 401(k), Roth IRA, and savings growth over time
- **Budget Tab** — per-category budgets with inline editing, savings auto-set to 10% of income
- **Persistent Storage** — all data saved to localStorage, nothing resets between sessions

---

## Built With

- HTML, CSS, JavaScript
- [Chart.js](https://www.chartjs.org/) — for charts and visualizations
- Google Fonts — Playfair Display + DM Sans

## How to Use

1. Download `velo.html`
2. Open it in any browser
3. Start adding transactions with the **+ Add** button

No installation. No terminal. No npm.
